# فروشگاه ۱

این ربات به صورت خودکار توسط ربات‌ساز ساخته شده است.

- نوع ربات: فروشگاهی
- امکانات: پرداخت کارت‌به‌کارت و تایید رسید، مدیریت محصول/خدمت، گزارش‌گیری، ارسال پیام گروهی، چند ادمین، درگاه پرداخت آنلاین، پنل مدیریت ساده، چندزبانه
- پشتیبانی: کشتیرانی

## پنل مدیریت داخل ربات
ادمین ربات با /start منوی مدیریت را می‌بیند و می‌تواند محصولات/خدمات، متن‌ها، سوال‌های فرم، کارت‌به‌کارت، لینک پرداخت و مرچنت زرین‌پال را تغییر دهد.

## نکته پرداخت آنلاین
مرچنت/API زرین‌پال داخل پنل ذخیره می‌شود. اتصال کامل callback و تراکنش خودکار بانکی فاز بعدی است.
